# Magasin Management App

Application mobile Flutter pour la gestion d'un magasin.

## Fonctionnalités

- Authentification JWT
- Gestion des produits
- Gestion des catégories
- Recherche de produits
- Calcul automatique du bénéfice
- Export PDF / Excel (à venir)

## Technologies

### Frontend
- Flutter
- Bloc
- SharedPreferences

### Backend
- Node.js
- Express.js
- JWT

### Base de données
- MySQL

## Structure

/frontend -> Application Flutter

/backend -> API REST Node.js

/database -> Scripts SQL
